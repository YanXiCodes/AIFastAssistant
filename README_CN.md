# AI Fast Assistant

[English](README.md) | 简体中文

> 一个轻量级的 AI 浮窗助手，让你在工作中随时快速提问，不打断当前流程

## 快速开始

### 下载安装
1. 前往 [Releases](https://github.com/yourusername/ai-fast-assistant/releases) 下载最新版本
2. 运行安装程序
3. 获取 DeepSeek API Key：[platform.deepseek.com](https://platform.deepseek.com)
4. 启动应用，输入 API Key

### 基本使用
- `Ctrl+Shift+Space` - 唤起窗口
- `Ctrl+Enter` - 发送消息
- `Esc` - 关闭窗口
- 输入 `/` - 快捷命令

## 主要功能

✅ 全局快捷键随时唤起  
✅ 选中文本快速查询  
✅ 8 种快捷命令模式  
✅ 历史记录快速访问  
✅ 代码语法高亮  
✅ 窗口透明度调节  
✅ Markdown 完整支持  
✅ 智能滚动控制  

## 为什么选择它？

- **快**：原生开发，启动秒开
- **轻**：无历史负担，内存占用低
- **省**：DeepSeek API 成本低至 GPT-4 的 1/50
- **专**：10+ 专业场景预设模板
